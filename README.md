# 여기에 제목

- 적응형 사이트(1280px*720px)
- Sass 사용.
- Router: Single Page Application 구현.
- Redux Toolkit: 메뉴 추가. 선택 옵션 관리.
- ReactBootstrap: 테이블, 캐러셀, 버튼, 모달 등을 활용한 UI 구현.
- styled component: 공통적으로 사용되는 요소 스타일 지정.
